# 🎮 Game of Life на React
[![Maintainability](https://qlty.sh/gh/lKoshil/projects/game-of-life-react/maintainability.svg)](https://qlty.sh/gh/lKoshil/projects/game-of-life-react)

Интерактивная реализация игры "Жизнь" Конвея с визуализацией, управлением скоростью и пресетами паттернов.

##  Стек

- **Frontend:** React 18, Vite
- **Стили:** CSS3 (glassmorphism)
- **Деплой:** Vercel

##  Как запустить локально

```bash
git clone https://github.com/lKoshil/game-of-life-react.git
cd game-of-life-react
npm install
npm run dev
